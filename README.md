#Machine Learning Based Diabetes Classification and Prediction
📌 Project Overview
This project focuses on predicting whether a person has diabetes using Machine Learning techniques.
It uses clinical and demographic data such as glucose level, BMI, age, and insulin levels to classify patients as diabetic or non-diabetic.

🎯 Objectives
Predict diabetes using ML algorithms

Compare performance of different models

Improve accuracy using preprocessing and feature selection

🧠 Algorithms Used
Random Forest

XGBoost

LightGBM

📊 Dataset
PIMA Indian Diabetes Dataset

Features include:

Pregnancies

Glucose

Blood Pressure

Skin Thickness

Insulin

BMI

Diabetes Pedigree Function

Age

⚙️ Technologies Used
Python

Scikit-learn

Pandas & NumPy

Matplotlib & Seaborn

Jupyter Notebook

🔄 Workflow
Data Collection

Data Preprocessing

Feature Selection

Model Training

Model Evaluation

Prediction

📈 Results
XGBoost achieved the best accuracy among all models

Random Forest also performed well with stable results
